Acerca de esta página

Maquetación CSS

Para conseguir una maquetación responsive con la menor cantidad de media queries y CSS he utilizado grid y flexbox en para maquetar la página.

  • Grid para la estructura general de la página
  • Flexbox para la maquetación de los elementos (header, footer, article...)

Elementos CSS3

He utilizado distintas características de CSS3

  • Variables para colores y gradientes utilizados en mas de una ocasión
  • Gradientes de color: en el fondo, y gradientes animados en el título y en el nav
  • Animación de subrayado del nav: se muestra al hacer hover sobre los enlaces
  • Animación de gradiente: cuando hacemos hover sobre el título y los elementos del nav
  • Filtros: drop-shadow para las imágenes

Javascript

Ha sido necesaria la utilización de javascript en estos casos:

  • Controlar si el menú hamburguesa se muestra o no, con un script que se dispara al cambiar el tamaño de la ventana
  • Evitar que las animaciones se inicien antes de que la página termine de cargar para evitar problemas de visualización en Chrome

Validadores

Tanto el HTML como el CSS han sido validados usando las herramientas del W3C
