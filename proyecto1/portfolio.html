Quizzical

Proyecto desarrollado en javascript, utilizando React. Utiliza la API gratuita de OpenTriviaDB para cargar preguntas nuevas cada vez que el usuario inicia una partida. Se pueden seleccionar el tipo de preguntas y la dificultad.

Discogs digger - En desarrollo

Proyecto desarrollado en javascript, utilizando node como backend, y NextJS (framework de React) para el frontend. El backend consiste en un servicio que hace llamadas a la API de Discogs para retornar al frontend un JSON con los resultados de su búsqueda, y con los videos asociados a cada release. Esto no es posible sin hacer varias llamadas a la API de Discogs, de ahí la necesidad de desarrollar este backend.

Se pretende facilitar una de las maneras de encontrar música nueva en discogs, ya que la página por defecto no permite mostrar los resultados de búsqueda y escucharlos en la misma página.
